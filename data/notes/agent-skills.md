# Agent Skills

Agent skills are reusable operating contracts that sit above raw tools and below product-level workflows.

A strong skill definition answers five questions:

1. What task shape should trigger the skill.
2. Which tools are allowed or expected.
3. What sequence or checklist the agent should follow.
4. What output quality bar applies.
5. What safety or escalation constraints exist.

In a mature agent platform, the skill layer prevents every task from becoming a fresh prompting exercise. Skills compress institutional knowledge into reusable task policies.
